# Proof of Concept

An Angular v19 Hello World application generated with Angular CLI and npm.

## Setup

Install dependencies:

```sh
npm install
```

## Development

Start the local development server:

```sh
npm start
```

The app is served by Angular CLI, typically at `http://localhost:4200/`.

## Build

Create a production build:

```sh
npm run build
```

Build output is written to `dist/`.

## Test

Run the default Angular test suite:

```sh
npm test
```

This project uses the Angular CLI Karma test setup, which requires a browser environment.
