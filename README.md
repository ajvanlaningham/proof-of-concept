# Proof of Concept

Proof of Concept is a clean Angular starter application generated with Angular CLI 19.2.26. It provides the default Angular project scaffold, npm scripts, and test setup for future product work.

## Prerequisites

- Node.js and npm

## Install

Install project dependencies:

```bash
npm install
```

## Development Server

Start the local development server:

```bash
npm start
```

Angular serves the app locally and reloads when source files change.

## Build

Build the application:

```bash
npm run build
```

Build output is written to the `dist/` directory.

## Scaffolding

This app was generated with Angular CLI. Use the Angular CLI to scaffold new code:

```bash
npm run ng generate component component-name
```

Other Angular artifacts can be generated the same way, including directives, pipes, services, classes, guards, interfaces, enums, and modules.
